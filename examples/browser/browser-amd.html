Using the browser version of Woodman with an AMD loader

This page uses an AMD loader to run the browser version of Woodman.

<script type="text/javascript" src="../../node_modules/requirejs/require.js"></script>
<script type="text/javascript" src="../../dist/woodman-browser-amd.js"></script>
<script type="text/javascript" src="browser-amd.js"></script>

The browser-amd.js file contains the main call to require and logs a few events to the console:

require(['woodman'], function (woodman) {
  var config = { ... };
  woodman.load(config, function (err) {
    if (err) throw err;

    var logger = woodman.getLogger('browser.main');
    logger.log('Welcome to Woodman');
    logger.info('This is an info message');
    logger.warn('This is a warning');
    logger.error('This is an error');
  });
});

Open your console to see the result!

Using the standalone version of Woodman in Web browsers

This page shows how to use the stand-alone version of Woodman in Web browser applications.

<script type="text/javascript" src="../../dist/woodman-browser.js"></script>
<script type="text/javascript" src="standalone.js"></script>

The stand-alone version exposes Woodman in window.woodman. The standalone.js file can access it directly:

var config = { ... };
woodman.load(config, function (err) {
  if (err) throw err;

  var logger = woodman.getLogger('browser.main');
  logger.log('Welcome to Woodman');
  logger.info('This is an info message');
  logger.warn('This is a warning');
  logger.error('This is an error');
});

Open your console to see the result!

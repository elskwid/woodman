Referencing the source code of Woodman from Web browsers

This example references Woodman's browser entry-point directly in the source code using require.js. In particular, it does not use a compiled version of Woodman.

<script type="text/javascript"
  src="../../node_modules/requirejs/require.js"
  data-main="source.js">
</script>

The source.js file contains the main call to require and logs a few events to the console:

require(['../../lib/woodman-browser'], function (woodman) {
  var config = { ... };
  woodman.load(config, function (err) {
    if (err) throw err;

    var logger = woodman.getLogger('browser.main');
    logger.log('Welcome to Woodman');
    logger.info('This is an info message');
    logger.warn('This is a warning');
    logger.error('This is an error');
  });
});

Open your console to see the result!
